# title
ILYSMIH by Kali Uchis interactive Lyric site

# project overview
The goal of this project was to create a stylized website using only words. The theme of the words we chose was "community." 

Before we began this project, we answered the question: What does community mean to you? What song or poem represents community to you and why? Based on this prompt, I decided to choose the song ILYSIMH by Kali Uchis. I chose this song because it was written about the love that she has for her newly born child and how he has changed her life. I thought that this represented community because children are an integral part of many communities. I also thought that the song is often mistaken for being a romantic love song, but I wanted to prove that community and love songs can be non-romantic as well. Community is not only found through romantic relationships but also through all types of relationships, whether with friends, family, or even pets.

# technical overview
My project is a multipage website that uses a series of animations to convey emotion and depth. I built this site using HTML and CSS only. My website goes from page to page through word links. For example, on page one, the word love is linked to page 2, so if you hover over the word and click on it, you will be taken to page 2. This goes for every page, including page 8, which loops back to page 1. I was able to learn how to make certain animations through online research and the help of AI tools such as Microsoft Pilot, which helps to explain how certain functions behave and how to get your website to work correctly. I also used Google Fonts to get my desired fonts for specific words and phrases. I also utilized lectures and information on Canvas to create my website as well.
